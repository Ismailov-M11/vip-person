# VIP Person — Web App

Платформа для сотрудничества бизнеса и блогеров (Узбекистан).
Интерактивный мобильный прототип. Чистый статичный фронтенд — **сборка не требуется**,
React и JSX выполняются прямо в браузере.

## Структура
```
index.html          — точка входа
app/                 — данные, UI-примитивы, экраны, роутер (.jsx + data.js)
frames/              — рамка iPhone
brand/               — логотип (SVG)
netlify.toml         — конфиг деплоя
```

## Запуск локально
Откройте любой статический сервер в этой папке (нельзя просто `file://`,
браузер заблокирует загрузку скриптов):

```bash
# вариант 1
npx serve .

# вариант 2
python3 -m http.server 8080
```
Затем откройте http://localhost:8080

## Деплой через GitHub → Netlify
1. Создайте репозиторий на GitHub и загрузите **содержимое этой папки**
   (важно: `index.html` должен лежать в корне репозитория).
2. На netlify.com → **Add new site → Import an existing project** → подключите репозиторий.
3. Настройки сборки оставьте пустыми:
   - **Build command:** *(пусто)*
   - **Publish directory:** `.`
   (Эти значения уже заданы в `netlify.toml`.)
4. **Deploy** — готово.

Либо просто перетащите эту папку в окно **Netlify → Sites → Drag & drop** —
сайт опубликуется без GitHub.
